### Переиспользуемые классы

- AssertUtils - клас утилит для различного рода шаблонных проверок, например на null входного параметра.
- Систему Gui из интерфейсов и абстрактного класса представлений и вью - IPresentable, IGuiView и GuiPresenter<TView>.
- В целом можно выделить шаблон для стратегии с результатом работы стратегии (применяется для стратегии хода). При помощи 
  факторизации его можно объединить с АТД Step, которая представляет стратегию поведения на конкретном шаге.
- AsyncInputHandle - класс реализации позволяющий работать с консольным вводом асинхронно, получая события при вводе.
