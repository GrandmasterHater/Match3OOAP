### Классы реализации

   Для построения игрового поля предполагается использовать двумерный массив. Для ведения истории ходов двусвязный список. 
Для хранения комбинации обычный список. Все перечисленные структуры данных предоставляются стандартными библиотеками, 
а значит выделение отдельных классов реализации не требуется.
