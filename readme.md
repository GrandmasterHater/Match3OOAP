#### Консольная матч 3

Ходы выполняются через ввод координат. 
Есть бонусы за создание линейных комбинаций и бонусы применяемые вречную, которые перемешивают элементы на поле.
Игра может завершаться по команде или автоматически, когда заканчиваются возможные ходы.

#### Запуск проекта

Требуется NET SDK 9.0.202 - https://dotnet.microsoft.com/en-us/download/dotnet/9.0

1. Загрузить папку с проектом.
2. Выполнить CLI команду из корневой папки репозитория: `dotnet run --project Match3OOAP\Match3OOAP\Match3OOAP.csproj --framework net9.0 -v q`
